## 👩🏼‍💻 Installation 👩🏼‍💻

1. **Clone the repository:**

    ```bash
    git clone https://github.com/rhyzJ/missionOne.git
    ```

2. **Install dependencies:**

    ```bash
    npm i
    ```

3. **Run the program:**

    ```bash
    npm run dev
    ```
